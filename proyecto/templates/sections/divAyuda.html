         1. Este sistema es el sistema de visualización de información de las condiciones nacionales básicas de China, la información proviene de datos oficiales del gobierno.

         2. Esta plataforma muestra los datos geoespaciales básicos de China, incluidos Hong Kong, Macao y Taiwán.

         3. De acuerdo con los requisitos gubernamentales pertinentes, estos datos son datos históricos y no se garantiza que se actualicen continuamente.

         4. En la etapa inicial de desarrollo, es inevitable que haya deficiencias y omisiones. Si hay algún defecto, entiéndalo.

         5. Agradecimiento especial: Instructor: Mora Navarro, Joaquin Gaspar (España); Miembro del equipo de I+D: Wu,Ruochen (China); Dujardin,Clément (Francia); Ducasse,Antoine (Francia)

         6. Para obtener información oficial más actualizada, visite la Red Nacional de Datos (El idioma del sitio web es el CHINO).

         7. Unidad de apoyo: Universitat Politècnica de València (UPV)

         1. 该系统是中国基本国情的信息可视化系统，信息来自官方政府数据。

         2. 该平台展示了中国的基本地理空间数据，包括了中国的港澳台地区。

         3. 根据政府的相关规定，所展示的数据均为历史数据，且无法保证持续更新。

         4. 此项目为开发初期阶段，不可避免地会存在缺陷和纰漏。 如有不足，请谅解。
